Every model used in this project has been acquired from cgtrader.

Dependencies and requirements.
Python 3.6 or above is required.
Numpy is also required to deal with the matrices, arrays and mathematical options on these and can be installed using 'pip install numpy'
Pygame must be installed. This can be done at https://www.pygame.org.
PyOpenGL must also be installed. This can be done at https://pypi.org/project/PyOpenGL/.
If using a mac a VM is required as the program does not run on Mac. On windows or Linux there should be no issues.
This virtual machine can be installed at https://cdimage.ubuntu.com/focal/daily-live/current/.
